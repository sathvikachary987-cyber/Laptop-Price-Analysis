# Laptop Price Analysis

## Domain
Data Analyst

## Project Overview
This project analyzes laptop specifications and prices to understand which features influence laptop pricing. 
The analysis focuses on data cleaning, exploratory data analysis (EDA), and visualization without using machine learning.

## Dataset Description
- Rows: 1275
- Columns: 23
- Source: Online laptop listings

## Features Included
- Company, Product, TypeName
- Screen Size, RAM, Weight
- Operating System
- Touchscreen, IPS Panel, Retina Display
- CPU & GPU Company
- Storage Type and Capacity
- Price (Euros)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- ydata-profiling

## Data Cleaning Steps
- Removed duplicate records
- Checked missing values (no missing data)
- Standardized text columns
- Converted numeric columns to proper data types

## Exploratory Data Analysis (EDA)
- Average price by company
- Laptop type distribution
- Weight vs price analysis
- Screen size vs price trend
- OS vs laptop price
- Touchscreen & IPS panel distribution
- GPU company vs laptop price

## Key Insights
- Premium brands like Razer, LG, and MSI have higher prices
- macOS laptops are more expensive than Windows & Linux
- NVIDIA GPUs are associated with high-end laptops
- Touchscreen and IPS displays are premium features

## Conclusion
Laptop pricing is mainly influenced by brand, GPU, OS, and display features. 
This analysis helps understand market trends and pricing patterns.
